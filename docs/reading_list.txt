== The 3 anchor benchmarks AERead integrates ==

[T1] TERMS-BENCH: Diagnosing LLM Negotiation Agents Beyond Deal Rate  [AXIS-1]
     Zhang, Zhang, Pappu, El, Blanchet, Athey, Liu, Zou
     arxiv 2605.13909 (https://arxiv.org/abs/2605.13909) (May 2026, Stanford)
     Live leaderboard: https://terms-bench.github.io (7-axis design precedent for AERead)
     Repo: github.com/zou-group/terms-bench (404 as of 2026-05-26; release expected within 1-3 months)
     → Bayesian-game framework for bilateral price negotiation with
       environment-as-verifier. 4-axis 6-metric design (SE+, AGR+/CSE+/FAGR-,
       BEtype, CritViol%). Six counterpart families × three regimes. Oracle
       is *simulator-relative* Bayes-optimal — explicitly NOT normative. KEY
       methodological contribution AERead generalizes: intervention-based
       oracle-gap decomposition (Δ_inf + Δ_unc + Δ_ctrl via base /
       oracle-posterior / revealed-type interventions). 13 LLM agents evaluated.
       Senior advising: Susan Athey + Jose Blanchet + James Zou; Alvin Roth
       early feedback. v2 D2 implementation path for AERead: wrap, don't
       rebuild. Coordination target: Erica Zhang.

[T1] Revealed Rationality: Label-Free Regularization from Representation
     Theorems  [AXIS-2]
     Isaiah Andrews (MIT Dept of Economics + NBER, sole author) — Feb 2026
     PDF in this repo: papers/pdfs/andrews_2026_revealed_rationality.pdf
     Published: economics.mit.edu/sites/default/files/2026-02/Revealed%20Rationality_1.pdf
     → THE academic foundation for AERead's Tier 4 commercial path + the
       Axis 2 mathematical anchor. Proves classical decision-theory
       representation theorems can be turned into LABEL-FREE, polynomial-
       time-computable continuous penalties for LLM training. Three
       instantiations: (1) de Finetti probabilistic coherence via Dutch-
       book LP — L(p) = max_b min_ωj Π(ωj); (2) Afriat preference rationality
       via CCEI (1 − CCEI ∈ [0,1]); (3) Echenique-Saito SARSEU joint
       beliefs+preferences via LP. Key §7 quote AERead adopts: "coherence
       is not sufficient for good behavior... so other training signals
       remain essential" — anchors §1.5.1 descriptive-vs-normative framing.
       Andrews §3 non-identification critique (representation theorems
       guarantee existence, not identification) anchors §1.5.6.5
       within-class identification framing. Sole-author + early-2026 MIT/NBER
       means widely circulated in LLM-economics community — cite extensively.

[T1] EconEvals: Benchmarks and Litmus Tests for Economic Decision-Making
     by LLM Agents  [AXIS-3 partial; reliability score adjacent to AXIS-2]
     Fish, Shephard, Li, Shorrer, Gonczarowski
     arxiv 2503.18825 (https://arxiv.org/abs/2503.18825) (March 2025, Harvard + Penn State)
     Repo: github.com/sara-fish/econ-evals-paper (open source)
     → Closest direct competitor at the Cat B-C parameter+outcome layer.
       Three economic environments (procurement / scheduling / pricing) via
       tool-use API; agent learns environment in context across 100 periods
       at 3 difficulty levels. Three litmus tests for tradeoff revelation:
       Patience vs Impatience (direct overlap with our B2), Efficiency vs
       Equality, Collusiveness vs Competitiveness. Each litmus emits LITMUS
       + RELIABILITY + COMPETENCY scores. Direct §1.5.7 judge-free anchor
       quote: "our litmus tests utilize stylized environments, which means
       the metrics that measure tradeoff responses are directly grounded in
       an underlying theoretical model." Models tested: Claude-3.5/Gemini
       1.5/GPT-4o/GPT-4.1/o4-mini/Gemini 2.5/GPT-5/Gemini 3 Pro Preview.
       Adoption-traction warning: 1 cite at 14 months despite Harvard PI —
       AERead's v0.5 plan is to WRAP EconEvals as Layer 3 D-dimensions
       (D4 procurement / D5 pricing / D6 scheduling) per layer3_candidates.md.

== 5-axis taxonomy prior-art anchors ==

[T1] Utility Engineering: Analyzing and Controlling Emergent Value Systems
     in AIs  [AXIS-3 primary anchor + adoption-playbook exemplar]
     Mazeika, Yin, Tamirisa, Lim, Lee, Ren, Phan, Mu, Khoja, Zhang, Hendrycks
     arxiv 2502.08640 (https://arxiv.org/abs/2502.08640)
     (Feb 2025, Center for AI Safety / CAIS / Hendrycks group)
     → AXIS-3 (Calibration) PRIMARY anchor — establishes that LLMs develop
       coherent utility functions at scale (Bradley-Terry MLE on paired
       comparisons; ~15 LLMs tested; fit quality rises monotonically with
       scale). AERead's Layer 2 identification methodology extends their
       Bradley-Terry framework to multi-class fits (KT prospect '79 / CRRA
       / CARA / CPT '92 / Kelly-Markowitz) per §1.5.6.5; Axis 3 calibration
       scoring uses Brier/CRPS per Gneiting-Raftery 2007 (strictly proper)
       with ECE for diagnostic visualization per Guo 2017. CRITICAL: Mazeika
       provides Layer 1 (existence) + Layer 2 (identification) evidence
       but NOT Layer 3 (predict-then-validate) — that's our extension.
       ALSO: 50 cites + NeurIPS 2025 + viral X thread in 15 months — closest
       reference adoption-playbook exemplar for AERead's launch.

[T1] When Agents Say One Thing and Do Another: Validating Elicited Beliefs
     from LLMs  [AXIS-5 primary anchor]
     Yamin, Tang, Cortes-Gomez, Sharma, Horvitz, Wilder
     arxiv 2602.06286 (https://arxiv.org/abs/2602.06286) (2026, CMU + Microsoft Research)
     → AXIS-5 (Meta-cognitive) PRIMARY anchor. Decision-theoretic framework
       eliciting BOTH probability judgments + decisions from an LLM and
       testing internal consistency: is there *any* near-rational utility
       function that rationalizes the action given the stated belief?
       Utility-free framing (no assumption on utility shape). Clinical-
       diagnosis tasks; frontier-model belief reports are imperfect
       summaries of decision-revealed information, but gap is small for
       strong models. The "is there *any* utility that rationalizes the data"
       framing is exactly AERead's Layer 2 identification logic + the
       stated-vs-revealed cross-check that addresses Zhu-Griffiths 2024's
       truthful-reporting caveat.

[T1] Dutch Books and Money Pumps: Rectifying Vulnerabilities in LLMs Through
     Rationality  [AXIS-5 + Tier 4 remediation]
     Chadwick, Kahng, Kipper — HAR 2025 workshop, Paris (Univ. of Rochester)
     PDF in this repo: papers/pdfs/chadwick_kahng_kipper_2025_dutch_books.pdf
     URL: https://ansonkahng.com/docs/papers/llmrationality.pdf
     → AXIS-5 (Meta-cognitive coherence-without-ground-truth) + SECOND
       academic foundation for AERead Tier 4 (inference-time remediation).
       MORE than Dutch-book / money-pump exploitation — proposes a
       CONSTRUCTIVE RATIONALITY LAYER: (1) probabilistic coherence via
       linear pooling + quadratic programming; (2) preference transitivity
       via Iterative Max Directed Cut (polynomial-time Kemeny approximation).
       Three IMDC variants with theorems comparing acyclic vs cyclic
       tournament graphs. Explicitly rebuts Goldstein's "LLMs cannot in
       principle be coherent" architectural argument. AERead's diagnostic
       page §6 "Counterfactual scenarios" can use IMDC as the "if transitivity
       were fixed" implementation. CRITICAL §Future Work quote AERead's §3
       thesis is built on: "(approximate) probabilistic coherence and
       transitivity of preferences are necessary conditions for rationality,
       they are arguably NOT sufficient" + "does a two-step refinement (first
       ordinal, then cardinal) outperform a single probability-based approach?"
       (the latter is precisely AERead's 3-layer pipeline stack-design question).

[T1] Incoherent Probability Judgments in Large Language Models  [AXIS-5]
     Zhu & Griffiths (CogSci 2024) — arxiv 2401.16646 (https://arxiv.org/abs/2401.16646)
     → AXIS-5 ready-made probe set. 24 probability-pair questions (12 weather
       + 12 politics) + probabilistic-identity battery (P(A) + P(¬A) − 1,
       P(A∧B) + P(A∧¬B) − P(A), etc.) any coherent agent must zero. Finding:
       4 SOTA LLMs exhibit human-like incoherence (inverted-U mean-variance
       matches Bayesian Sampler model). Theoretical claim: incoherence may
       stem from autoregressive training as implicit Bayesian inference —
       implies fine-tuning corrections may have a ceiling. CRITICAL footnote
       AERead addresses via Yamin et al. 2026 stated-vs-revealed cross-check:
       "LLMs were assumed to demonstrate truthful reporting of latent event
       probabilities" — AERead does NOT take stated confidence at face value.

[T1] Probabilistic Coherence, Logical Consistency, and Bayesian Learning:
     Neural Language Models as Epistemic Agents  [AXIS-2 + AXIS-5 + Tier 4]
     Betz & Richardson (PLOS ONE 2023) — doi 10.1371/journal.pone.0281372
     (https://doi.org/10.1371/journal.pone.0281372)
     → FOURTH academic foundation for AERead Tier 4 (self-supervised
     consistency training on synthetic structured corpora). The orthogonal-
     to-Qiu mechanism: no oracle teacher needed; the model self-trains
     toward a probabilistically coherent + logically consistent belief
     system. Coined the "Rationality Hypothesis" framing AERead
     operationalizes. CRITICAL §Future Research item 3 AERead's diagnostic
     page is the direct response to: calls for "improved diagnostic tools"
     with "high-resolution representations of an agent's probabilistic
     belief system" relating inferential structure of training data to
     current model behavior. Caveat: experiments on small T5 + synthetic
     corpora; frontier-scale not yet demonstrated.

== Diagnostic-and-correction literature (the §3 thesis cluster) ==

[T1] The Emergence of Economic Rationality of GPT  [AXIS-2; §7.A1 ceiling + §1.5.5 framing-sensitivity]
     Chen, Liu, Shan, Zhong — PNAS 2023
     arxiv 2305.12763 (https://arxiv.org/abs/2305.12763) (Lingnan + Tsinghua + HKUST + NUS)
     → GPT-3.5-Turbo on 25-task budget allocation × 4 domains × 100 reps =
       10K tasks. Rationality measured via GARP. GPT outperforms 347-person
       US-representative human sample. CCEI > 0.997 mean across domains —
       the source of AERead's §7.A1 CEILING-EFFECT warning (GARP may not
       differentiate at frontier-model level — extend dimensions / more
       observations needed). KEY NUANCE: rationality robust to demographic
       settings BUT BREAKS under different price framing OR discrete choice
       setting — direct empirical anchor for §1.5.5 context-conditional
       axiomatization. Dual-purpose anchor: ceiling-effect warning AND
       framing-sensitivity evidence.

[T1] Economic Rationality under Specialization  [AXIS-2; §1.5.5 canonical anchor]
     Wen 2025 — arxiv 2501.18190 (https://arxiv.org/abs/2501.18190)
     (Tsinghua Shenzhen, ShuiDe Wen)
     → §1.5.5 canonical anchor for context-conditional axiomatization.
       Extends Chen et al. 2023 by running budget-allocation + risk experiments
       on persona-specialized GPT agents (biotech expert, economist) vs
       generic base. Finding: specialization DEGRADES economic rationality —
       more GARP violations, lower CCEI, larger deviations under high-risk
       conditions. Coins "rationality shift." This is the SMOKING-GUN
       empirical evidence for AERead's §1.5.5 thesis: persona-conditioning
       is a measurable rationality dimension, not an assumption to control
       for. v0 Cat C/D use cases include persona-specialized prompts where
       this effect should be reproducible.

[T1] Bayesian Teaching Enables Probabilistic Reasoning in LLMs  [AXIS-5; Tier 4 third academic foundation]
     Qiu, Sha, Allen, Kim, Linzen, van Steenkiste
     arxiv 2503.17523 (https://arxiv.org/abs/2503.17523)
     (Nature Communications 2026; MIT + Meta + DeepMind + UBC + Vector + Google + NYU)
     → THIRD academic foundation for AERead Tier 4 (supervised fine-tuning
       on normative-model traces). LLMs plateau after 1 round of feedback
       (~37-40%); Bayesian Assistant reaches 81% after 5 rounds. KEY
       CONTRIBUTION: fine-tune on Bayesian Assistant TRACES (reasoning
       under uncertainty), not correct-answer traces. Bayesian model is
       better teacher than oracle because it teaches strategy, not just
       conclusion. Generalizes to new tasks (hotel + web shopping). Also
       Axis 5 metacog remediation precedent.

[T1] Thinking Fast and Slow in Large Language Models  [AXIS-2; §1.5.6 scaling narrative]
     Hagendorff, Fabi, Kosinski — arxiv 2212.05206 (https://arxiv.org/abs/2212.05206)
     (Nature Computational Science 2023; Stuttgart + UCSD + Stanford)
     → CRT (Cognitive Reflection Test) + semantic illusions on GPT-1 →
       ChatGPT-4 family. 50 bespoke task versions to prevent contamination
       (200 tasks total) + 500 human Prolific participants. Two findings:
       (1) GPT-1 → GPT-3 scaling makes MORE human-like intuitive errors
       (System 1 emerges with scale); (2) ChatGPT-3.5/4 avoid the traps EVEN
       WHEN PREVENTED FROM CHAIN-OF-THOUGHT (System-1-like generation
       becomes more accurate without explicit System-2 deliberation).
       Precursor to Tak et al. 2026's reasoning-helps story. Together
       Hagendorff + Tak tell the 4-stage scaling narrative for §1.5.6
       measurement-model pluralism.

[T1] Benchmarking the Rationality of AI Decision Making via Transitivity  [AXIS-2]

       AERead's §1.5.6 measurement-model pluralism formalizes — 4-6

[T1] Rationality Check! Benchmarking the Rationality of Large Language

[T2] LLM Economicus? Mapping the Behavioral Biases of LLMs via Utility Theory  [AXIS-3]

[T2] Sparks of Rationality: Do Reasoning LLMs Align with Human Judgment
     and Choice?  [AXIS-2; reasoning-mode evidence]
     Tak, Banayeeanzade, Bolourani, Bahrani, Chaubey, Karimireddy, Schwarz, Gratch
     arxiv 2601.22329 (https://arxiv.org/abs/2601.22329) (Jan 2026, USC / ICT)
     → Reasoning-enabled LLMs on (i) classical axioms + (ii) behavioral-econ
       domains (Allais, framing, certainty effects). Reasoning mode shifts
       models toward EU-maximizing answers and AWAY from human-like biases —
       dual-process pattern. Direct evidence for §1.5.1 descriptive-vs-
       normative split: SAME model tuned by thinking-budget toward either
       rational-axiom compliance OR human-like bias replication. Cite in
       §1.5.1 + in §7.B1-B3 to justify reporting thinking-mode-on/off
       conditions side by side.

[T2] QEDBENCH: Quantifying the Alignment Gap in Automated Evaluation of
     University-Level Mathematical Proofs
     Gonzalez et al. (Yale-led 51-author benchmark)
     arxiv 2602.20629 (https://arxiv.org/abs/2602.20629) (Feb 2026)
     → DIRECT empirical anchor for §1.5.7 judge-free-by-design claim.
       Measures LLM-as-judge alignment gap on math proofs and finds +0.18
       to +0.36 mean score inflation across 4 of 7 frontier judge models.
       7-judge × 5-solver matrix over 1000+ proofs with 1000+ hours of
       expert human evaluation as ground truth + dual-rubric design. Adopt
       the N×M matrix evaluation pattern. Reject their "closer-to-human =
       better" normative framing (§1.5.1 rejects this).

[T2] Alignment Revisited: Are Large Language Models Consistent in Stated

[T2] PTCBENCH: Benchmarking Contextual Stability of Personality Traits in

[T2] GuruAgents: Emulating Wise Investors with Prompt-Guided LLM Agents

[T2] Can an LLM Learn Preferences from Choice Data?

== Econometric / Athey lineage (structural-estimation framing) ==

[T3] SHOPPER: A Probabilistic Model of Consumer Choice with Substitutes and
     Complements
     Ruiz, Athey, Blei
     → Susan Athey's closest work to AERead's setup. Probabilistic model
       inferring consumer preferences from observed shopping baskets. 13F
       is structurally a multi-category consumer-choice problem. Useful
       bridge to Erica via Athey co-authorship.

[T3] Counterfactual Inference for Consumer Choice Across Many Product
     Categories
     Athey et al. — arxiv 1906.02635 (https://arxiv.org/abs/1906.02635)
     → Counterfactual reasoning over consumer choices. AERead's implied-
       conviction inversion is structurally a counterfactual question.

[T3] Inverse Reinforcement Learning using Revealed Preferences and Passive
     Stochastic Optimization
     Krishnamurthy — arxiv 2507.04396 (https://arxiv.org/abs/2507.04396)
     → Title is dead-on for AERead's methodology though not by a TERMS-Bench

       Reference if probed "is this IRL or revealed preferences?"

== Calibration + scoring-rule foundations (Axis 5 / §6 reproducibility) ==

[T3] On Calibration of Modern Neural Networks  [AXIS-5]
     Guo, Pleiss, Sun, Weinberger (ICML 2017) — arxiv 1706.04599 (https://arxiv.org/abs/1706.04599)
     → Field-standard for predicted-confidence vs realized-accuracy gap.
       Introduces ECE + reliability diagrams. Temperature scaling as 1-param
       post-processing fix. AERead adopts ECE + reliability diagrams for
       the diagnostic page; NOT for leaderboard ranking (ECE is not
       strictly proper — see Gneiting-Raftery 2007).

[T3] Strictly Proper Scoring Rules, Prediction, and Estimation  [AXIS-5]
     Gneiting & Raftery (JASA 2007)
     PDF: sites.stat.washington.edu/raftery/Research/PDF/Gneiting2007jasa.pdf
     → Foundational theory of proper scoring rules. Brier, logarithmic,
       spherical, quadratic, CRPS. Calibration-vs-sharpness joint
       decomposition. AERead's headline Axis 5 score uses Brier (discrete)
       / CRPS (continuous) per this paper; ECE only for diagnostic
       visualization. Methodological hook: a leaderboard score that models
       are incentivized to maximize MUST be strictly proper or gameable.

== Decision-theory foundations (non-vNM axiomatizations, for v2 D1) ==

[T3] Knightian Decision Theory, Part I
     Bewley (2002) — Decisions in Economics and Finance 25
     → Relaxing COMPLETENESS — agents may refuse to rank some pairs.
       Cite when methodology paper discusses non-vNM axiomatizations.

[T3] Maxmin Expected Utility with Non-Unique Prior
     Gilboa, Schmeidler (1989) — Journal of Mathematical Economics 18
     → Foundational paper for ambiguity-averse decision-making. Drops
       INDEPENDENCE. Relevant when agent context is Ellsberg-like genuine
       ambiguity.

[T3] Regret Theory: An Alternative Theory of Rational Choice Under Uncertainty
     Loomes, Sugden (1982) — Economic Journal 92
     → Foundational paper for TRANSITIVITY-violating but coherent
     preferences. Cycling can be rational if outcomes are evaluated
     relative to counterfactual regret. Reference for §1.5.5 risk
     discussion.

[T3] Prospect Theory: An Analysis of Decision under Risk
     Kahneman, Tversky (1979) — Econometrica 47
     → KT '79 — empirically-grounded relaxation of INDEPENDENCE that anchors
       Category B. Already implicitly cited via §7.B1.

[T3] Advances in Prospect Theory: Cumulative Representation of Uncertainty
     Tversky, Kahneman (1992) — Journal of Risk and Uncertainty 5
     → CPT '92 — cumulative prospect theory with rank-dependent weighting.
       AERead Layer 2 multi-class fits distinguish KT '79 from CPT '92 as
       separate functional classes per §7.C1.

[T3] A Theory of Anticipated Utility
     Quiggin (1982) — Journal of Economic Behavior & Organization 3
     → Rank-dependent utility theory; alternative to KT prospect theory.
       Useful when comparing axiomatization families.
